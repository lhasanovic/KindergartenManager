KindergartenManager - Help

Before Running

The default administrator password is: 11112222

On the first run, the app uses default teacher and children data. You’ll need to manually enter the data you wish to use.

During form filling, light-green indicates acceptable field values, whereas light-pink indicates invalid values in the field.

Home Screen

You can change the app’s language by clicking the corresponding flag. After that, the app will shut down and you will have to open it again manually.

Click the corresponding button based on which panel you wish to open.

Teacher

If you are a teacher, click the Teacher button and enter your first name and ID in the corresponding fields before clicking Confirm.

Parent

If you are a parent, click the Parent button, enter your first name and ID, and click the Confirm button.

Admin

Ukoliko ste administrator, pritisnite na dugme Admin te unesite administratorsku šifru, prije nego pritisnete Potvrdi.

Teacher Panel

The table contains basic data about children registered in your class. Click on any child to select it.

If you want to change the current child activity, click the Change activity button, select an activity from the dropdown menu, and click OK.

If you want to add a diary entry, click the Add diary entry button. You can select the child from the table beforehand or you can select it from the dropdown menu which will appear. You will need to enter the date, time, and child activity. The description is optional. Click OK to confirm.

Finally, for diary-related child options, click the View diary button. Diary entries appear in a table, with indicated date, time, activity, and description. Here, you can still add a diary entry the same way or edit an existing one. It’s possible to delete a diary entry by clicking Delete diary entry -> OK.

To filter the diary based on the time of the activity, use the choice box in the upper part of the window. The return to the home screen is possible by clicking the home button in the upper left corner.

Parent Panel

The parent panel is simple and it contains the following information:

  • Current child activity
  • Teacher info
  • View diary button

Diary view is immutable because parents should not be able to change the diary.

Administrator

The admin panel contains child and teacher tables. To hire a teacher, click the Hire a teacher button, fill in the form, and click OK.

To fire a teacher, choose one from the table and click Fire teacher -> OK.

To edit a teacher, click the Edit teacher button, change some data, and click OK.

To register a new child, click the Register a child button. Fill in the form and click OK. To change the child data, use the Edit child button. Both forms are identical, only the editing form already contains data about the selected child.

To delete a child, click Delete child -> OK.

To delete the admin password, click the Change password button. You’l need to enter the old password and the new one in the correspodning fields. The password must contain between 8 and 16 characters!

Teacher report and Children report buttons open the corresponding reports. Returning to the home screen is possible by using the home button in the upper-left corner.
